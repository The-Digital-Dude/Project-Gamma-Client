## ًںڑ€ How It Works

### 1. Dynamic Sitemap (App Router)
The `src/app/sitemap.ts` file generates a dynamic sitemap that:
- Automatically includes all static routes
- Supports dynamic routes (like blog posts) when implemented
- Sets appropriate priority and change frequency for different page types
- Uses your site configuration from `src/config/site.ts`

### 2. Static Sitemap Generation
The `next-sitemap` package generates additional sitemap files during build:
- Creates `sitemap.xml` and `robots.txt`
- Handles complex routing patterns
- Excludes private/admin routes

- `/subscription-*` (Payment flows)

## ًں“ٹ Sitemap Structure

The implementation creates:
1. **Dynamic sitemap** at `/sitemap.xml` (generated by Next.js)
2. **Static sitemap** files in `public/` folder (generated by next-sitemap)
3. **Robots.txt** file pointing to your sitemap
